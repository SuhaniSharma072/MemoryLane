Memory lane is a website using HTML, CSS, and JavaScript which is like a scrapbook in which users can add pictures and information of their past hackathons they attended, allowing them
to record their experiences and reflect on projects they’ve built, skills they’ve learned, and people they’ve met, all in one personalized and interactive digital space. The main challenge 
I faced while coding this was how to code it so that it saves. I was able to fix this by learning how to add localStorage to the javascript code.
